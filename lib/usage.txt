# oe-cloud-cli


Usage:
oe starter-app [options]

Options:
-h, --help # Print generator's options and usage Default: false

Description:
Creates an oe based starter application

Options:
--skip-install: Skips the automatic execution of bower and npm
after scaffolding has finished.
--s: alias --skip-install

Example:
oe starter-app [--skip-install] [--silent]
